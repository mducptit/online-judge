## Overview

Homepage: [http://oj.ptitteam.com](http://oj.ptitteam.com)

Based on [DMOJ](https://dmoj.ca/).

Supported languages: 

- Assembly (x64)
- AWK
- C
- C++03 / C++11 / C++14 / C++17
- Java 11
- Pascal
- Perl
- Python 3

Support plagiarism detection via [Stanford MOSS](https://theory.stanford.edu/~aiken/moss/).
